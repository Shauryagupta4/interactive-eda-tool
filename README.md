# Interactive EDA Tool

An interactive Exploratory Data Analysis (EDA) web application built using **Python, Pandas, Streamlit, Matplotlib, and Seaborn**.

This tool allows users to upload any CSV dataset and instantly perform basic data analysis, data quality checks, correlation analysis, and visualizations through an easy-to-use web interface.

## Features

### Dataset Upload

* Upload any CSV file
* Instant dataset loading and preview

### Data View

* View complete dataset inside the browser
* Interactive dataframe display

### Summary Statistics

* Generate descriptive statistics using Pandas
* Mean, Median, Standard Deviation, Min, Max, Quartiles

### Data Schema Analysis

* Total number of records
* Dataset dimensions (rows & columns)
* Column data types
* Missing value count
* Duplicate row detection

### Missing Value Detection

* Identify missing values across the dataset
* Easy visualization of null records

### Dynamic Visualizations

Users can generate:

* Bar Charts
* Pie Charts
* Line Charts
* Histograms

Features:

* Dynamic column selection
* Automatic validation of column types
* Error handling for unsupported datasets

### Correlation Matrix

* Numerical column correlation analysis
* Heatmap visualization using Seaborn
* Helpful for identifying relationships between variables

## Installation

### Clone the Repository

```bash
git clone https://github.com/Shauryagupta4/interactive-eda-tool.git
cd interactive-eda-tool
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run app.py
```
